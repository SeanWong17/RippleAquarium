# Local Decor Models

Place a local pineapple house GLB here as:

```text
pineapple-house.glb
spongebob-patrick.glb
```

The scene loader keeps the full GLTF scene and original materials, then scales it
to the aquarium floor. If the file is missing, the app uses the procedural
fallback house in `src/decor/pineapple-house.js`.

Current local-test source:

- "Bfbbr-SpongeBob Pineapple House" by SMF Features Developed From Cheryl Hill
- https://sketchfab.com/3d-models/bfbbr-spongebob-pineapple-house-4e2d36c5f95645448b44af409432ae82

- "NASB2 - SpongeBob and Patrick" by SMF Features Developed From Cheryl Hill
- https://sketchfab.com/3d-models/nasb2-spongebob-and-patrick-717a58577d554b86802162db847c7f13
